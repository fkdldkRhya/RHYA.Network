RHYA.Network
===

![RHYA.Network Logo](rhya-network-logo.png)

## RHYA.Network
* Product distribution
  - RHYA.Network에 연결되어있는 모든 서비스 배포 기능
    + 파일 다운로드
    + 자동 업데이트
    + 공지사항
    + 서비스 관리
* Account management
  - RHYA.Network Auth.V1 (통합 계정 관리 시스템)
    + 회원가입
    + 로그인
    + 비밀번호 찾기
    + 2단계 인증
    + 계정 차단
    + OAuth 인증
* Web service management
  - 페이지 접근 차단
  - IP 차단

## Development and deployment environment
